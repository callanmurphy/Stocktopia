Getting Started

1. Click the download button in the top right corner of this webpage to get started!

2. Unzip the library package to obtain the stocktopia.js and stocktopia.css files

3. Add the two stocktopia files to your working directory

4. Connect pages to the library by adding the following lines in an HTML file:

    <link rel="stylesheet" type="text/css" href="path/to/stocktopia.css">

    <script type="text/javascript" src="path/to/stocktopia.js"></script>


Docs v1.0

Stock

new Stock(ticker, exchange, info);


Property Default Type Description
ticker null String Ticker name for the stock
exchange null String Exchange name for the stock
info 5 randomly assigned times/prices Dictionary Dictionary of time-price pairs for a stock's daily data. Randomly generated if not provided
times [] Array List of one-day time data for the stock. Created by constructor from 'info'
prices [] Array List of one-day price data for the stock (same length as 'times'). Created by constructor from 'info'
stats [filled by constructor] Array 5 length stats array in the order: Change ($), Change (%), Open, Close, Range

Stock.table(id, colour, data)

Displays a stock table at HTML DOM id


Property Default Type Description
id stock.ticker + '-table' String DOM element ID
colour rgb(217,62,154) HTML colour Table title colour
data 5 randomly assigned times/prices Dictionary Dictionary of time-price pairs for a stock's daily data. Randomly generated if not provided

Stock.chart(id, colour, data)

Displays a stock chart at HTML DOM id


Property Default Type Description
id stock.ticker + '-chart' String DOM element ID
colour rgb(217,62,154) HTML colour Chart colour
data 5 randomly assigned times/prices Dictionary Dictionary of time-price pairs for a stock's daily data. Randomly generated if not provided

Stock.addData(newTimes, newPrices)

Add more data to an existing stock - length of newTimes and newPrices must match


Property Default Type Description
newTimes [] Array New times to be added to stock.times
newPrices []] Array New prices to be added to stock.prices
